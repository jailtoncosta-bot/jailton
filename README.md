# 🌾 CultivaEconomia - Calculadora de Economia da Agricultura Familiar

Este é um projeto de site estático desenvolvido para o **GitHub Pages**, projetado especificamente para ajudar agricultores familiares a calcularem a economia gerada ao utilizarem insumos orgânicos/compostagem própria em vez de fertilizantes químicos tradicionais, além da venda direta.

O objetivo é conscientizar e fornecer uma ferramenta prática de tomada de decisão para o pequeno produtor.

## 🚀 Funcionalidades
- **Cálculo em Tempo Real:** Descubra a economia mensal e anual estimada.
- **Interface Responsiva:** Funciona perfeitamente em celulares, tablets e computadores.
- **Design Temático:** Cores inspiradas na natureza e na agricultura sustentável.
- **Hospedagem Gratuita:** Feito puramente em HTML5, CSS3 e JavaScript, ideal para o GitHub Pages.

## 📂 Estrutura de Arquivos
- `index.html`: Estrutura de conteúdo e formulários do site.
- `style.css`: Estilização, layout e responsividade.
- `script.js`: Lógica matemática para o cálculo da economia.

## 🛠️ Como Publicar no GitHub Pages
1. Crie um repositório público no seu GitHub (ex: `meu-site-agricultura`).
2. Adicione os arquivos `index.html`, `style.css`, `script.js` e este `README.md` nele.
3. Vá em **Settings** (Configurações) do seu repositório.
4. No menu lateral esquerdo, clique em **Pages**.
5. Em **Build and deployment**, mude a Branch para `main` (ou `master`) e a pasta para `/ (root)`.
6. Clique em **Save**. Em alguns minutos, seu site estará online no link fornecido pelo GitHub!
